## How to Add Topics

1. Go to: https://github.com/Kobeep/devops-macos-workstation
2. Click ⚙️ (gear icon) next to "About" on the right sidebar
3. Add topics from the list above
4. Click "Save changes"

This will make your repo discoverable through GitHub search and topic pages!
