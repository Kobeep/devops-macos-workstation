## Guidelines to Follow

1. **Check existing entries** - Make sure it's not already listed
2. **Follow formatting** - Match the existing format exactly
3. **Add to correct category** - Read category descriptions carefully
4. **Include description** - Keep it concise (1-2 lines)
5. **Alphabetical order** - Most awesome lists require this
6. **One PR per list** - Don't submit to multiple lists in one PR
